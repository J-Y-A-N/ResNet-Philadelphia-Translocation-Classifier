About this program:

This is a t(9;22) translocation(also known as Philadelphia Translocation) classifer trained using Google's Teachable Machine platform.


To use this program:


  1. Crop out the images of chromosomes 9 and 22 from a regular karyotype image
  2. Create a new image of size 60*60 that contains chromsome 9 in the left half and chromosome 22 in the right half(examples of images can be found in the image folder)
  3. Drag the preprocessed image onto the canvas
  4. Click the "Classify Image" button
